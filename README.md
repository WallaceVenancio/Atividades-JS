# 🟨 Atividades JavaScript 🚀

Este repositório contém atividades práticas de JavaScript desenvolvidas durante meus estudos na [Alura](https://www.alura.com.br/), com o objetivo de consolidar lógica de programação, manipulação do DOM e estruturação de pequenos projetos web.

## 📁 Estrutura

- `alugames/` — Locadora de jogos de tabuleiro online
- `sorteador-numeros/` — Sorteador de números
- `carrinho-compras/` — Carrinho de compras de produtos
- `ingresso/` — Sistema de compra de ingressos para shows
- `amigo-secreto/` — Sorteador de amigo secreto
- `...` — Novas atividades serão adicionadas regularmente

## 🛠️ Tecnologias utilizadas

- **JavaScript (JS)**
- **HTML e CSS** (fornecidos pela plataforma ou professor; foco principal no JS)

## 🎯 Objetivo

- Praticar lógica de programação
- Aprender a interagir com o DOM
- Desenvolver pequenas aplicações web
- Evoluir como futuro Desenvolvedor Front-End
